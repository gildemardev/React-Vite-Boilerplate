# React + TypeScript + Vite

Esse starter foi pensado para facilitar o início de um
projeto react rapidamente, garantindo uma base sólida para SPA's.

## Instalando o Boilerplate

```bash
git clone
```

```bash
npm install
```

```bash
npm run dev
```
